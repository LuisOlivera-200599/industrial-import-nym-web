Control rápido del catálogo comercial

Desde aquí puedes agregar productos, editar información, cambiar el estado de stock y eliminar registros. Todo lo que guardes aparecerá en la página pública de productos usando esta demo con almacenamiento local.

Modo demo activo

Los datos se guardan en este navegador. Para producción real, luego se puede conectar con Firebase, Supabase o una base de datos propia.

1. Agrega

Completa nombre, marca, categoría, imagen, stock y descripción.

2. Guarda

El producto se registra y queda listo para mostrarse en el catálogo.

3. Edita

Usa el botón del lápiz para corregir datos o actualizar stock.

4. Elimina

Borra productos que ya no deben aparecer en la gestión local.
